本仓库提供一种用于雷达抗干扰回波恢复的 ODE-SDE 预测校正生成框架。该方法面向强压制干扰、欺骗干扰以及压制-欺骗复合干扰场景，将受干扰雷达回波恢复建模为复数时频域中的条件传输与局部校正问题，从而在保持目标相干结构的同时实现低步数、低延迟的生成式恢复。
1、方法概述
传统扩散模型通常从无结构高斯噪声先验出发，并依赖多步反向采样，容易带来较高推理开销，也可能破坏雷达复数回波中的相位一致性。为解决这些问题，本项目采用两阶段预测校正框架：
ODE Predictor：通过条件速度场学习从受干扰状态到纯净目标状态的平均传输方向，实现单步确定性粗恢复，并获得目标结构锚点。
Brownian-Bridge SDE Corrector：以 ODE 粗估计为锚点，在局部邻域内对残余干扰、结构伪影和相位误差进行细粒度校正。
复数域约束：在复数时频表示和时域回波上约束波形一致性与相位结构，提升目标相干结构保持能力。
2、主要特性
支持 ISRJ 欺骗干扰、CSNJ 压制干扰以及 ISRJ+CSNJ 复合干扰场景。
提供传统 STFT 方法、U-Net、CS-NET 等基线方法用于对比。
同时评估 SINRIF、RMSE、Corr、鲁棒性曲线和推理效率。
采用 ODE 单步预测与 SDE 单步校正，兼顾恢复精度和低延迟推理。
